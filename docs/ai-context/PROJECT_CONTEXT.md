## Archived body (pre-April 2026)

KeyPilot is a modular SaaS platform designed for real estate professionals.

The platform will eventually support:

- lead capture
- open houses
- showings
- CRM functionality
- transaction management
- analytics
- automation

The first product module being built is:

### Open House Lead Capture & Showing Management

This module allows real estate agents to:

• create property records  
• create open house events  
• capture visitor sign-ins (tablet or QR)  
• automatically generate leads  
• generate follow-up email drafts  
• produce seller summary reports  

Future platform modules planned:

- FarmTrackr
- ClientKeep (CRM)
- DealForge (transaction management)
- InsightDeck (analytics)
- AutoPilot (automation engine)

The system architecture must support adding these modules later without rewriting the system.
